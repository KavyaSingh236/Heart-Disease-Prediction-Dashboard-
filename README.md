# Heart Disease  Prediction Dashboard  
### Predictive Analytics with Machine Learning in Tableau

---

## Overview

This project presents an interactive heart disease prediction dashboard built using clinical cardiovascular data.

The objective was to develop a predictive analytics solution that enables real-time risk estimation based on patient health indicators. A classification model was integrated within the dashboard to generate probability-based predictions dynamically.

The project demonstrates how machine learning can enhance traditional data visualization by embedding predictive capabilities directly into an interactive analytics environment.

---

## Problem Statement

Heart disease remains one of the leading causes of mortality worldwide.  
Early risk assessment based on measurable health indicators can support preventative clinical decision-making.

This dashboard simulates patient risk scoring using commonly recorded cardiovascular features.

---

## Project Workflow

### 1. Data Understanding
- Imported cardiovascular dataset into Tableau
- Examined feature distributions and target variable (CHD)
- Selected clinically relevant predictors

### 2. Model Development
- Constructed feature matrix from clinical variables
- Trained a classification model using scikit-learn
- Generated both binary classification and probability outputs
- Integrated model inference within the dashboard for real-time predictions

### 3. Interactive Simulation
- Created parameter controls for clinical inputs
- Enabled users to modify patient characteristics
- Returned dynamic risk probability based on selected inputs

### 4. Dashboard Design
- Structured layout for intuitive interpretation
- Incorporated visual indicators for predicted risk
- Built a simulation-focused analytical experience
